# 6 KHUYẾN MÃI MẪU – DÙNG ĐỂ CẬP NHẬT TRONG ADMIN

Dùng thông tin bên dưới để thêm/sửa từng khuyến mãi trong **Admin → Khuyến mãi** (hoặc chạy file SQL tương ứng).

---

## 1. Xong mùa thi – Quẩy mùa hè (Học sinh/Sinh viên)

- **Mã khuyến mãi:** SUMMER25
- **Tên:** Xong mùa thi Quẩy mùa hè!
- **Mô tả:** Áp dụng dành cho khách hàng thành viên thuộc đối tượng Học sinh - Sinh viên. Tặng 1 nước ngọt miễn phí khi giao dịch từ 2 vé xem phim trở lên.
- **Loại:** Tặng sản phẩm (FREE_ITEM)
- **Giảm (%):** 0
- **Giảm (₫):** 0
- **Đơn tối thiểu (₫):** 0
- **Giảm tối đa (₫):** 0
- **Ngày bắt đầu / kết thúc:** Tùy bạn (vd: hôm nay → +90 ngày)
- **Số lần dùng (tổng):** 2000 (hoặc để trống = không giới hạn)
- **Lần/người:** 1
- **Trạng thái:** Bật

## 2. Vé 58K/62K – MoMo

- **Mã khuyến mãi:** MOMO58
- **Tên:** BETA VÉ RẺ MOMO MUA LIỀN!
- **Mô tả:** 58.000₫/vé Thứ 2 - Thứ 6; 62.000₫/vé Thứ 7 - Chủ nhật. Áp dụng khi thanh toán qua Ví MoMo.
- **Loại:** Giảm tiền cố định (FIXED_AMOUNT)
- **Giảm (%):** 0
- **Giảm (₫):** 42000 (giảm còn ~58K so với giá gốc, tùy giá vé của bạn)
- **Đơn tối thiểu (₫):** 100000
- **Giảm tối đa (₫):** 0
- **Ngày bắt đầu / kết thúc:** Tùy bạn
- **Số lần dùng (tổng):** 1000
- **Lần/người:** 2
- **Trạng thái:** Bật
