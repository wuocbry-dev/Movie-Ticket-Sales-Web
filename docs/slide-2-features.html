Introduction

This slide presents the complete list of key features of the Movie Ticket Sales System, organized by the four user roles that interact with the platform: Customer, Cinema Staff, Cinema Manager, and System Admin.

🔴 Customer Features

Movie Discovery — Customers can browse the full movie catalog, filtered by Now Showing and Coming Soon. Each movie page includes detailed information such as genre, duration, synopsis, cast, trailers, and audience ratings.

Seat Selection & Booking — The system provides a real-time interactive seat map for each showtime. Customers select available seats, which are temporarily locked during the booking flow to prevent double-booking.

Concession Ordering — Alongside ticket booking, customers can add food and drinks from the concession menu specific to each cinema location. This adds convenience and increases per-transaction revenue.

Promotions & Loyalty — Customers can apply discount codes at checkout and earn loyalty points with each purchase. Accumulated points can be redeemed on future bookings to encourage repeat usage.

🔵 Staff Features

Ticket Check-In — Cinema staff can validate customer tickets at the entrance by entering or scanning a booking code. The system confirms validity in real time, reducing fraud and manual errors.

Payment Processing — Staff can handle on-site payments for walk-in customers. Managers have access to full transaction histories and can initiate refunds where necessary.

🟢 Management & Admin Features

Cinema & Hall Management — Managers can configure and manage multiple cinema branches, individual halls, and their seating layouts from a single admin interface.

Showtime Scheduling — Managers create and update showtimes for each movie per cinema. The system automatically detects scheduling conflicts to prevent overlapping sessions in the same hall.

Account Management — System admins control all user accounts across the platform, including assigning roles (customer, staff, manager) and managing staff assignments to specific cinemas.

Movie Management — Admins can add, edit, or remove movies from the catalog, including full metadata: genres, cast, poster images, trailers, and release date information.

Concession Catalog — Admins manage all concession categories and menu items, and can assign different menus to different cinema locations based on availability.

Admin Dashboard — A central overview page gives administrators a live snapshot of bookings, revenue trends, active users, and system-wide activity — supporting data-driven decisions.

Access Control Summary

All features are protected by role-based access control (RBAC) using JWT authentication. The four roles — Customer, Cinema Staff, Cinema Manager, and System Admin — each have clearly defined permissions, ensuring that sensitive operations are restricted to authorized users only.

Tech Stack

The platform is built with React on the frontend for a dynamic, responsive UI; Spring Boot on the backend with a RESTful API architecture; MySQL as the relational database; Docker for containerized deployment; and JWT for secure, stateless authentication across all roles.
